Lightning Privacy Policy

This document is for our Privacy Policy. By using Lightning, you agree to the document. Do not copy/plagarize/steal this document.



User Data

The only data that is stored is your warnings in a server, and your user ID for our economy system.



Guild Data

Guild data is stored. As in the guild ID, and some features with true and false like Flag Translations, Ticketing, auto moderation, etc. etc.



Channel Data

Channel ID's are stored. They are stored for ticket logging, automod logs, and the suggestion system.



Category Data

Category ID's are stored. They are stored for ticket logging.



Data

Your data is not viewed by anyone. You may request for your data to be removed. If you request your user data to be deleted, it will delete all stats, member data, and user data. If you requst guild data to be deleted it will remove all guild data.
